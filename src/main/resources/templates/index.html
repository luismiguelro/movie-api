Acerca de la API de Películas

La API de Películas es desarrollada con Spring Boot y desplegada en fly.io. Permite obtener información sobre películas, detalles por ID, añadir reseñas y obtener reseñas de una película en particular. ¡Explora y disfruta!

Endpoints Disponibles
  • Obtener todas las películas
  • Obtener información de una película por ID
  • Añadir una reseña a una película
  • Obtener todas las reseñas de una película por ID
